# Qual e-mail deve ser linkado para solicitar desconto de exame relativo a Eventos?

Para eventos selecionados da Microsoft, os participantes podem ser elegíveis a um desconto no exame de certificação da Microsoft. Dependendo do evento, o e-mail que deve inserir para verificar os descontos elegíveis pode ser diferente. Utilize a tabela abaixo para ver se o evento é elegível e as instruções apropriadas sobre como solicitar o seu desconto:

| Eventos | Instrução sobre qual e-mail inserir |
| --- | --- |
| [Microsoft Ignite September 2019](/learn/certifications/microsoft-ignite-free-certification-exam-offer?WT.mc_id=msignitethetour2019_akawwlflag_-email-event) | Email que usou para se registar no evento |
| [Microsoft Ignite the Tour 2019-2020](/learn/certifications/microsoft-ignite-free-certification-exam-offer) | Email que utilizou para se inscrever no evento |
| [Microsoft Build May 2020](/learn/certifications/microsoft-build-cloud-skills-challenge-2020-free-certification-exam-offer) (Evento Virtual) | Email que utilizou para se registar no evento |
| Microsoft Virtual Training Day - Azure Fundamentals | Email que usou para se inscrever no evento |
| Microsoft Virtual Training Day - Dynamics Fundamentals | Email que usou para se inscrever no evento |
| Microsoft Virtual Training Day - Power Platform Fundamentals | Email que usou para se inscrever no evento |
| [Microsoft Ignite Setembro 2020](/learn/certifications/microsoft-ignite-cloud-skills-challenge-2020-free-certification-exam) (Evento Virtual) | Email que utilizou para completar o Cloud Skills Challenge em Learn (Email no Perfil Learn) |
